# El Caso Mafe

Escape room familiar estático para María Fernanda Leal Pulido.

## Uso rápido

1. Abrir `index.html` en el navegador.
2. Ingresar el código de acceso al expediente.
3. Imprimir las pistas desde el botón del sitio.
4. Revisar `docs/guia-organizador.html` para montaje, respuestas y orden de escondites.

## Archivos

- `index.html`: juego principal.
- `css/styles.css`: estilos y versión imprimible.
- `js/app.js`: lógica del escape room y quiz.
- `assets/`: íconos e ilustración del tablero.
- `docs/guia-organizador.html`: guía reservada para la familia.

No usa Firebase, PWA, dependencias externas ni conexión a internet.
